# スキル：CEPポテンシャルスコア算出

**使用するエージェント：** insight-analyst / content-strategist  
**入力：** CEP候補リスト・各指標データ  
**出力：** CEPポテンシャルスコア表・優先順位

---

## 算出手順

### STEP 1：6指標のデータを収集する

| 指標 | 収集方法 | スコア範囲 |
|-----|---------|---------|
| CEP連想総数 | 消費者調査・想起調査 | 実数（人数） |
| CEP浸透率 | 消費者調査（過去1ヶ月での経験率） | 0〜100% |
| 経験頻度 | 購買ログ・アンケート（月あたり回数） | 実数 |
| 習慣強度 | 購買データ・競合調査（1〜5段階） | 1〜5 |
| WTP | 価格調査（そのCEPで出せる金額） | 実数（円） |
| LTV | CRMデータ（離脱までの平均累計金額） | 実数（円） |

### STEP 2：各指標を0〜10に正規化する

```
正規化スコア = (実測値 - 最小値) / (最大値 - 最小値) × 10

※ 比較対象：同一調査内の全CEP候補の値
```

### STEP 3：重み付けしてポテンシャルスコアを算出する

```
ポテンシャルスコア =
  CEP浸透率(正規化)   × 0.25  ← 市場規模の代理指標
  + 経験頻度(正規化)  × 0.20  ← 接触機会の多さ
  + 習慣強度(正規化)  × 0.20  ← 競合への対抗しやすさ（低習慣強度 = 参入チャンス）
  + WTP(正規化)       × 0.20  ← 収益ポテンシャル
  + LTV(正規化)       × 0.15  ← 長期収益ポテンシャル
```

※ 習慣強度は「低いほど参入チャンス」のため、スコアを反転させる（10 - 正規化スコア）

### STEP 4：CEPマップに配置する

横軸：自社MPen（メンタルペネトレーション）
縦軸：競合MPen

各CEPを4象限にプロットして戦略方針を確定する：
- 第一象限（競争的）：自社も競合も強い
- 第二象限（劣後的）：自社が弱く競合が強い
- 第三象限（未開発）：自社も競合も弱い
- 第四象限（維持的）：自社が強く競合が弱い
