# Jobs（ジョブ定義・無消費実態）

市場に存在するジョブ（顧客が特定の状況でなし遂げたい進歩）を蓄積するフォルダです。

## 保存対象
- 機能的・感情的・社会的ジョブの定義
- 無消費層の実態（なぜ正規の解決策を選ばないか）
- 間に合わせの対処策（顧客が自分で工夫している代替手段）
- ジョブポテンシャル検証スコア（購入意向×新奇性）

## ファイル命名規則
`YYYY-MM-DD_[テーマ]_job-definition.md`
