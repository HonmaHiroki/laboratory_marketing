# CEP Map（カテゴリーエントリーポイント）

ブランドが想起されうる購買文脈（CEP）を蓄積・管理するフォルダです。

## 保存対象
- 発見されたCEP候補と優先度
- CEPとブランドの連想強度
- 競合がカバーしていないCEP
- ライフイベント・季節性と連動するCEP

## ブランド成長とCEPの関係

| 指標 | 説明 | 重要な段階 |
|------|------|---------|
| MPen（メンタルペネトレーション） | 1つ以上のCEPでブランドを認知している人の割合 | 小〜中規模ブランド |
| NS（ネットワークサイズ） | 1人あたりのCEP連想数 | 成長期ブランド |
| MMS（メンタルマーケットシェア） | 全CEPを通じた想起シェア（≒MPen×NS） | 全段階のKGI |

## ファイル命名規則
`YYYY-MM-DD_[ブランド/カテゴリー]_cep-map.md`
