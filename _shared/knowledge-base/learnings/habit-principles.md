# Learnings / Habit Principles（習慣形成・維持の確定知識）

習慣形成・維持に関して検証済みの原則を保存するフォルダです。

## 初期登録：組織の初期原則

```markdown
---
確定日: 初期登録
ステータス: 参照原則（要検証）
タグ: 習慣, ライフイベント, ハビットスリップ
---

### 原則1：習慣形成段階と維持段階で報酬設計を変える
- 形成段階 → 継続報酬（毎回同じ確実な報酬）でオートパイロット化
- 維持段階 → 変動報酬（予測不能なタイミングの報酬）でエンゲージメント維持

### 原則2：ライフイベントは競合習慣を崩す絶好機
引越し・結婚・就職・出産などのライフイベント時は、消費者が目的志向モードに切り替わり、
新しいブランドを受け入れやすくなる。この瞬間を狙うコミュニケーションが有効。

### 原則3：ハビットスリップを防ぐには既存習慣との互換性が鍵
新商品を「いつでも使える」と伝えるより、
「〇〇（既存の習慣）の後に使う」と具体的な習慣文脈を提示する方が定着率が高い。

### 原則4：ダウトストックが大きなCEPを狙う
競合の期待-効用ギャップが大きな文脈（競合習慣が弱い場所）は
自社ブランドへの乗り換えが起きやすい。
```
