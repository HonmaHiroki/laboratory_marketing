# Learnings / Communication（コミュニケーション戦略の確定知識）

コミュニケーション設計に関して検証済みの原則を保存するフォルダです。

## 初期登録：組織の初期原則

```markdown
---
確定日: 初期登録
ステータス: 参照原則（要検証）
タグ: コミュニケーション, ブランデッドリーチ, 未顧客
---

### 原則1：ブランデッドリーチが意味のある広告リーチの本質
「広告を見た」だけでなく「広告を見て、かつブランドと正しく結びついた」リーチが重要。
8割のTVCMはブランデッドリーチを達成できていないとされる。

### 原則2：広告から購買まで3つの断絶ポイントを防ぐ
1. 「広告→ブランド」：広告を見てもブランドを識別できない
2. 「文脈→属性」：購買文脈でブランドの商品属性が想起されない
3. 「広告→店頭」：広告を見ても店頭でブランドを見つけられない

### 原則3：未顧客へのコピーはメンタルシミュレーションを重視する
「実際に使っているところがイメージできる」具体的な場面描写が、
未顧客の変化への抵抗感を和らげる。

### 原則4：ロッサー・リーブスの誤謬を避ける
ファン（ロイヤル顧客）に有効な施策・メッセージを、未顧客に転用しても効果がない。
「ブランドが好き」という交絡因子が未顧客には存在しないため。

### 原則5：広告は機能説明より「文脈的な手がかり」を伝える
「何ができるか」より「どんな場面で、どんな感情的報酬が得られるか」を
CEPと紐付けて伝えることで、ブランドと購買文脈の記憶連合が強化される。
```
