# Campaign Results / By Audience（顧客タイプ別施策結果）

ロイヤル顧客・競合顧客・未顧客それぞれへの施策結果を蓄積するフォルダです。

## 重要原則

**ロイヤル顧客で有効だった施策が未顧客に有効とは限らない（ロッサー・リーブスの誤謬）。**
顧客タイプ別に結果を分けて記録し、誤った転用を防ぐこと。

## サブフォルダ構成

```
by-audience/
├── loyal/          ロイヤル顧客向け施策結果
├── competitor/     競合顧客向け施策結果
└── non-customer/   未顧客向け施策結果（←最も重視）
```

## ファイル命名規則
`YYYY-MM-DD_[顧客タイプ]_[施策名]_result.md`
