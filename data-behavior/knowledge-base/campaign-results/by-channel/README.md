# Campaign Results / By Channel（チャネル別施策結果）

チャネルごとの施策結果を蓄積し、チャネル×CEP×顧客タイプの勝ちパターンを見つけるフォルダです。

## チャネル一覧
- `tv-video/` : TV・動画広告（マス認知・MPen拡大向け）
- `seo-content/` : SEO・コンテンツ（高関与・検討段階向け）
- `sns/` : SNS（日常文脈との結びつき向け）
- `email/` : メール（ナーチャリング・習慣形成向け）
- `in-store/` : 店頭・売り場（購買直前の習慣発動向け）
- `ec-d2c/` : EC・D2C（習慣スクリプト離脱向け）

## ファイル命名規則
`YYYY-MM-DD_[チャネル]_[施策名]_result.md`
