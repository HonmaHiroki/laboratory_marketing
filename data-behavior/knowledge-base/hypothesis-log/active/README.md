# Hypothesis Log / Active（検証中の仮説）

現在検証中の仮説を管理するフォルダです。

## ファイル命名規則
`YYYY-MM-DD_[仮説タイトル略称]_hypothesis.md`

## 昇格・移動ルール
- 検証で支持 → `../verified/` に移動（支持回数を記録）
- 検証で棄却 → `../rejected/` に移動（失敗の構造分析を追記）
- 180日以上検証されない → `../rejected/` にアーカイブ

---

## 対象顧客タイプとN1像
- 対象：ロイヤル顧客 / 競合顧客 / 未顧客

- 未顧客にも有効と考える根拠：

## 検証方法
