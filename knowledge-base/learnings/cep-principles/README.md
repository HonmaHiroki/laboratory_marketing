# Learnings / CEP Principles（CEPマネジメントの確定知識）

CEPに関して検証済みの勝ちパターン・原則を保存するフォルダです。

## 初期登録：組織の初期原則

組織が参照すべき初期原則です。
今後の検証を通じて更新・追加していくこと。

---

```markdown
---
確定日: 初期登録
ステータス: 参照原則（要検証）
タグ: CEP, ブランド成長
---

### 原則1：第一想起はブランド成長の「結果」であり「先行指標」ではない
第一想起を目指すのではなく、多くのCEPで考慮集合に「入ること」を目指す。
1等賞より「入賞すること」が重要。

### 原則2：ブランド規模で見るべき想起指標が変わる
- 小〜中規模ブランド → 助成想起・MPen（フロア効果のため純粋想起は動かない）
- 成長期ブランド → ネットワークサイズ（NS）・文脈想起
- 大規模ブランド → 純粋想起・シェアオブマインド（SOM）

### 原則3：純粋想起が動き始めるのは助成想起が50%を超えてから
純粋想起の成長を待つのではなく、まず助成想起の50%突破を目指す。

### 原則4：未顧客の記憶変化は助成想起に現れる
ブランド成長の源泉は未顧客にあるため、未顧客の助成想起変化を追跡する。
```
