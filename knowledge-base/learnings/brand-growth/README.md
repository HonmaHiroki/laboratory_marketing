# Learnings / Brand Growth（ブランド成長の確定原則）

ブランド成長全体に関わる確定知識を保存するフォルダです。

## 初期登録：組織の初期原則

```markdown
---
確定日: 初期登録
ステータス: 参照原則（要検証）
タグ: ブランド成長, 未顧客, アベイラビリティ
---

### 原則1：ブランド成長の源泉は未顧客にある
ヘビーユーザーのロイヤリティ強化より、未顧客・ライトユーザーの浸透率拡大の方が
ブランド成長への寄与が大きい（ダブルジェパディの法則）。

### 原則2：メンタル×フィジカルアベイラビリティの両輪が必要
- メンタルアベイラビリティ（記憶に入ること）だけでは購買につながらない
- フィジカルアベイラビリティ（買える場所にあること）がなければ想起が無駄になる
- 最も直接の利用経験が記憶構造を強化する

### 原則3：ブランド成長のフロー
① MPen増加（1つ以上のCEPでブランドを認知する人が増える）
② NS増加（1人あたりのCEP連想数が増える）
③ 助成想起が50%超（純粋想起が動き始める）
④ 純粋想起増加
⑤ 第一想起（ブランド成長の「結果」）

### 原則4：「やや満足」は離反の入口、「大満足」だけがリピート・紹介につながる
「やや満足」の顧客がリピートや紹介をする確率は極めて低い。
「大満足」の顧客を増やすことがサービス事業成長の核心。

### 原則5：顧客・従業員・事業の三者を犠牲にしないサービス設計
どこかが自己犠牲を払うサービスモデルは持続しない。
顧客の事前期待×従業員の事前期待が交差するクロスポイントを設計することが重要。
```
